## üöÄ About the Project

**MuscleMan** is a robust, full-stack application designed to streamline and manage all aspects of a personal fitness regime or a small gym's operations. Leveraging the stability and power of the Microsoft .NET ecosystem, it provides a reliable, scalable, and secure platform for tracking progress, managing client data, and planning workouts.

*This project aims to be the single source of truth for your fitness journey.*

## ‚öôÔ∏è Installation and Setup

Follow these steps to get your local development environment up and running.

### Prerequisites

Ensure you have the following software installed:
* [Visual Studio](https://visualstudio.microsoft.com/downloads/) (Community or Professional edition)
* The required **.NET Framework** version (Check the project properties in Visual Studio).

### Setup Steps

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Yugkalathiya/MuscleMan.git](https://github.com/Yugkalathiya/MuscleMan.git)
    ```
2.  **Open the Solution:**
    Open the `MuscleMan.sln` file in Visual Studio.
3.  **Restore NuGet Packages:**
    Right-click the solution in the Solution Explorer and select "Restore NuGet Packages."
4.  **Database Configuration:**
    * Locate the connection string in your configuration file (`Web.config` or similar).
    * Update the connection string to point to your local SQL Server instance or ensure LocalDB is configured correctly.
5.  **Run the application:**
    Press **F5** in Visual Studio or click the **Start** button to build and launch the application in your default browser.
